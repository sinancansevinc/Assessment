## assessment-api

A project for Assessment to Rise Technology


### Tech stack
- .NET Core 5
- MsSQL
- Git
- Redis

### Database Schema

#### Contacts
- Id
- TypeId
- ContactContent
- PersonId

#### Persons
- UUID
- Name
- Surname
- Company

#### ContactTypes
- Id
- Type


#### Routes API
You can access routes with using swagger.

#### NOTE
- You need to change connection string on app settings.
- Redis must be installed on your computer.
